# Image assets

Organize static images by purpose so forked projects stay tidy.

| Folder | Purpose | Example |
|--------|---------|---------|
| `heroes/` | Full-width hero backgrounds | `hero-home.svg` |
| `destinations/` | Destination card photos | `bali.svg` |
| `testimonials/` | Traveler avatar crops | `avatar-sarah.svg` |
| `icons/` | Feature and UI icons (SVG) | `globe.svg` |
| `og/` | Open Graph / social share images | `default.svg` |

Reference images from CMS content using paths from `public/`, e.g. `/images/heroes/hero-home.svg`.

Demo assets in this repo are simple SVG placeholders — replace with your own photos for production.
